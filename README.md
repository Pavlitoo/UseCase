### **Білет №1**  
**1. Способи обробки зрізу борта. Технологічна послідовність обробки відрізними підбортами:**  
**Способи:**  
- Обробка підбортом  
- Обробка обтяжкою  
- Обробка окантовкою  
- Обробка швом вподгин  

**Технологічна послідовність:**  
1. Підготувати деталі підборту  
2. Обробити внутрішній зріз підборту  
3. Скласти підборт з бортом лицьовими сторонами всередину  
4. Прострочити шов з'єднання  
5. Підрізати припуски шва  
6. Вивернути на лицьову сторону  
7. Виправити кант  
8. Закріпити підборт до внутрішніх швів  

**2. Основні групи асортименту тканин для спідниць:**  
- Костюмні (габардин, діагональ, твід)  
- Плащові (щільні, водовідштовхувальні)  
- Трикотажні полотна  
- Джинсові  
- Легкі (шифон, креп-жоржет)  

---

### **Білет №2**  
**1. Технологічна послідовність обробки розрізу та шліци у спідниці:**  
1. Намітити місце розташування  
2. Укріпити кінці розрізу машинними стібками  
3. Обробити краї обметувальним швом  
4. Підігнути краї згідно з припусками  
5. Застрочити підгин потайним/прямим швом  
6. Обробити кути розрізу вручну  
7. Приутюжити готовий розріз  

**2. Тканини для штанів:**  
- Джинсові (денім)  
- Костюмні (вовняні, напіввовняні)  
- Спортивні (еластичні, дихаючі)  
- Літні (льон, бавовна, віскоза)  
- Зимові (щільні вовняні)  

---

### **Білет №3**  
**1. Способи обробки горловини у виробах без коміра. Технологічна послідовність обробки підкрійною обшивкою:**  
**Способи:**  
- Підкрійною обшивкою  
- Косою бейкою  
- Окантовкою  
- Швом вподгин  
- В'язаною резинкою  

**Технологічна послідовність:**  
1. Викроїти обшивку за формою горловини  
2. Обробити зовнішній зріз обшивки  
3. Накласти обшивку на горловину лицьовими сторонами всередину  
4. Прострочити шов з'єднання  
5. Підрізати припуски швів  
6. Надсікти припуски на закруглених ділянках  
7. Вивернути обшивку на виворітну сторону  
8. Виправити шов і приутюжити  
9. Закріпити обшивку до плечових швів  

**2. Тканини для суконь:**  
- Легкі (шифон, креп-жоржет, батист)  
- Костюмні (офісні, класичні)  
- Трикотажні  
- Святкові (атлас, оксамит, парча)  
- Змішані (універсальні)

### **Білет №4**  
**1. Види застібок у спідниці. Спосіб обробки застібки тасьмою-блискавкою:**  
**Види застібок:**  
- На тасьму-блискавку  
- На гудзики  
- На кнопки  
- На липучку  

**Способи обробки блискавки:**  
1. Потайний (блискавка не видна ззовні)  
2. Накладний (блискавка на поверхні тканини)  
3. В шов (вшивається безпосередньо в шов)  

**2. Тканини для халатів:**  
- Махрові (банні)  
- Фланель (теплі домашні)  
- Батист/ситець (літні)  
- Велюр (зимові)  
- Трикотаж (повсякденні)

### **Білет №5**  
**1. Способи обробки верхнього зрізу спідниці. Технологічна послідовність обробки пришивним поясом:**  
**Способи:**  
- Пришивним поясом  
- Еластичною тасьмою  
- Обшивкою  
- Швом вподгин  

**Технологічна послідовність:**  
1. Викроїти пояс з припусками  
2. Продублювати пояс флізеліном  
3. Обробити короткі зрізи поясу  
4. Скласти пояс навпіл, приутюжити  
5. Приметати пояс до верхнього зрізу спідниці  
6. Прострочити з'єднувальний шов  
7. Підігнути вільний зріз поясу  
8. Настрочити пояс на спідницю  

**2. Тканини для блуз:**  
- Сорочкові (поплін, батист)  
- Шовкові (креп-де-шин, атлас)  
- Віскозні (драпіровані)  
- Льняні (літні)  
- Синтетичні (зносостійкі)

### **Білет №6**  
**1. Модель жіночих брюк та алгоритм підготовки до першої примірки:**  
**Опис моделі:**  
- Силует: прямий  
- Довжина: до щиколотки  
- Застібка: блискавка у лівому бічному шві  
- Кишені: прорізні в бічних швах  
- Обробка низу: підгин  

**Алгоритм підготовки:**  
1. Перевірити якість викроювання  
2. Перенести мітки з лекал на тканину  
3. Обметати зрізи деталей  
4. Зметати основні шви (середні, бічні, шагові)  
5. Зметати виточки  
6. Провести вологу теплову обробку  
7. Підготувати виріб до примірки  

**2. Тканини для дитячого одягу:**  
- Натуральні (бавовна, льон)  
- Трикотажні полотна  
- Фланель (нижня білизна)  
- Джерсі (повсякденний одяг)  
- Махрові (банні халати)

### **Білет №7**  
**1. Види застібок у штанах. Технологічна послідовність обробки застібки-блискавки з відрізним відкоском:**  
**Види застібок:**  
- На блискавку з відкоском  
- На гудзики  
- На кнопки  
- Комбінована  

**Технологічна послідовність:**  
1. Викроїти деталь відкосу  
2. Обробити зріз відкосу  
3. Приметати блискавку до зрізу застібки  
4. Прострочити блискавку з лівого боку  
5. Накласти відкіс на блискавку  
6. Прострочити відкіс по контуру  
7. Виконати закріплювальну строчку  
8. Приутюжити  

**2. Прикладні матеріали для спідниць:**  
- **Прикладні:** дублерин, флізелін, підкладка  
- **Нитки:** поліестерові, бавовняні, армовані  
- **Фурнітура:** блискавки, гудзики, кнопки  
- **Правила підбору:** відповідність кольору, типу тканини, призначенню

### **Білет №8**  
**1. Види виточок. Технологічна послідовність обробки. Терміни:**  
**Види виточок:**  
- Нагрудні  
- Талієві  
- Плечові  
- Ліктьові  

**Технологічна послідовність:**  
1. Перенести лінії виточок  
2. Зметати виточки від широкої частини  
3. Прострочити, зводячи строчку нанівець  
4. Закріпити кінці виточок  
5. Приутюжити  

**Визначення термінів:**  
- **Наметування:** тимчасове з'єднання ручними стібками  
- **Пришивання:** постійне прикріплення деталей  
- **Запрасовування:** закріплення згину прасуванням  

**2. Матеріали для штанів:**  
- **Прикладні:** клейова прокладка, корсажна стрічка  
- **Нитки:** №40-50 (основні), №60-80 (декоративні)  
- **Фурнітура:** блискавки, гудзики, пряжки

### **Білет №9**  
**1. Види кокеток. Технологічна послідовність обробки:**  
**Види кокеток:**  
- Пришивні  
- Накладні  
- Суцільнокрійні  
- Фігурні  

**Обробка пришивних кокеток:**  
1. Обробити зріз кокетки  
2. Зібрати основну деталь  
3. Приметати деталь до кокетки  
4. Прострочити шов  
5. Обробити припуски  
6. Заутюжити шов  

**Обробка накладних кокеток:**  
1. Обробити зрізи кокетки  
2. Наметати кокетку на деталь  
3. Настрочити декоративною строчкою  
4. Видалити наметку  
5. Приутюжити  

**2. Вимоги до швейних ниток. Поліестерові нитки:**  
**Вимоги:**  
- Міцність і еластичність  
- Рівномірність товщини  
- Стійкість до стирання  
- Відповідність кольору  

**Види ниток:**  
- За сировиною: натуральні, штучні, синтетичні  
- За структурою: кручені, комплексні, текстуровані  

**Поліестерові нитки:**  
- Висока міцність та еластичність  
- Стійкість до хімічних речовин  
- Не дають усадки  
- Універсальні

### **Білет №13**  
**1. Види комірів. Технологічна послідовність обробки:**  
**Види комірів:**  
- Відкладні  
- Стійкі  
- Стояче-відкладні  
- Коміри-шалі  
- Коміри-хомути  

**Обробка відкладного коміра:**  
1. Продублювати нижній комір

5. Приметати до горловини  
6. Прострочити шов  

**Обробка коміра-стійки:**  
1. Продублювати одну деталь  
2. Зметати по верхньому зрізу  
3. Прострочити, вивернути  
4. Приметати до горловини  
5. Прострочити з двох сторін  
6. Обробити внутрішній зріз  

**2. Матеріали для блуз:**  
- **Прикладні:** флізелін, дублерин  
- **Нитки:** тонкі (№60-80), шовкові  
- **Правила:** відповідність товщині тканини, кольору

**Способи:**  
- Пришивною манжетою  
- Швом вподгин  
- Еластичною тасьмою  
- Обшивкою  
- Фігурним зрізом  

**Технологічна послідовність:**  
1. Викроїти манжету  
2. Продублювати манжету  
3. Обробити короткі зрізи  
4. Скласти манжету навпіл  
5. Зібрати низ рукава
